Ferramentas

voltar*

Atualmente, existem diversas ferramentas para a criação de código html, se tornando assim uma dúvida muito grande para muitos desenvolvedores. Ferramentas Html, ou editores html, são ferramentas usadas para a criação de páginas web, como essa que você está acessando por exemplo. Porém o mais usado, principalmente por sua facilidade de manuseio, diversas extensões para ajudar no desenvolvimento e ainda por cima totalmente gratuito é o VsCode.

Outro método que pode ajudar MUITO no desenvolvimento de um site é a utilização da ferramenta do seu próprio navegador para manipular os estilos e ver como a sua página Web funciona, podendo assim pensar em uma melhor forma de poder estar chegando no resultado pretendido e deixar os problemas mais visiveis para soluciona-los em seguida.

Estrutura do HTML

voltar*

A estrutura do HTML engloba o head e o body, ou seja, tudo que irá aparecer e não aparecer para o usuário. Diante disso, podemos compreender que o html é dividido em duas partes, sendo eslas o < Head > e o < body >, tudo que se encontra dentro do "head" irá ser carregado primeiro antes de transmitir para o usuário o design da página como qualquer outra informação. Já o body é tudo o que irá aparecer para os usuários, sendo toda aquela parte front end de um site, sendo textos, imagens, vídeo, audio, etc. E pra completar temos o próprio html que consiste tudo da página.

Falando sobre Tags

voltar*

Tags são as definições de formatação que designamos para a máquina entender e assim será convertida em uma formatação html, podendo explicar detalhe por detalhe como será o corpo da nossa página. Dentre os tipos de tags:

< i > Formata a letra para italico;
< strong > Formata a letra para negrito;
< u > Formata a letra para sublinhado;
< input/> Adiciona uma caixa para inserir dados (tem que especificar qual o atributo type="");
< img /> Permite anexar imagens para a página;

Atributos

voltar*

Atributos são propriedades que uma tag pode ter, adicionando elementos e características daquela tag em específico. São exemplos de Atributos: id="" , class="" , type="" , style="" , src="" , href="" , width="" , height="" , etc...

Textos

voltar*

Textos como o próprio nome já diz, são tags que tem a finalidade de dar sentido para a página, adicionando letras de diferentes tamanhos e com seus respectivos atributos. Existem vários tipos de tags para formatação de texto, sendo elas:

  • Tags: "h1", "h2", "h3", "h4", "h5" e "h6" (mudança de tamanho do h1 "maior" para o h6 "menor"); 😎
  • Tag: "p" consiste em um paragrafo; 😁
  • Tag: "Blockquote" é uma situação, observação sobre determinado assunto.; 🔮
  • Tag: "Mark" é um marca texto, similar aos marcas textos físicos; 📝
  • Tag: "Sup" formata uma 'Elevação', usados normalmente para números como 24 ou até mesmo em textos; ✍
  • Tag: "Small" como o próprio nome já diz em inglês, formata um texto deixando-o pequeno; 🤏
  • Tag: "Hr" cria uma linha entre o conteúdo da página, como esses que estão presentes nesse site; 😄

Listas Ol/Ul

voltar*

Listas são tags que são normalmente usadas para o desenvolvimento de Listas na página, podendo essas ser dois tipos: Ol (Lista Ordenada) ou Ul (Lista NÃO Ordenada). Ol são usadas quando é necessário colocar dados em que sua órdem de apresentação importe, vindo assim seguida de sua respectiva numeração. Já a 'não ordenada' (Ul) não tem uma órdem certa, vindo na formatação seguidas de 'bolinhas'.

Exemplo de lista Ol (Ordenada):


  1. Olá! Sou uma lista ordenada número 1 😃
  2. Olá! Sou uma lista ordenada número 2 🤠
  3. Olá! Sou uma lista ordenada número 3 😝

Exemplo de lista Ul (Não Ordenada):


  • Olá! Sou uma lista não ordenada 👧
  • Olá! Sou uma lista não ordenada 👱‍♂️
  • Olá! Sou uma lista não ordenada 🤶

Links

voltar*

Links é uma tag que permite transportar um usuário que está em seu web site para outros, como também permite além de leva-los para onde achar melhor, existe um atributo que possibilita esse meio de transporte sem que os usuários saiam de sua página. O nome desse parametro é target="_blank".

Muitos também usam o a para direcionar, porém o significado da tag < a > é "ancora".

✅ atributos de link: target="_blank" abre o link em uma outra aba do navegador;
✅ title="" coloca um titulo e aparece para o usuario quando deixa o mouse em cima do link;
